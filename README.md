## 📌 功能介紹
### 🔹 前台
- 瀏覽商品清單與詳細資訊
- 商品分類與系列篩選
- 加入商品至購物車
- 下單購買流程
### 🔹 後台
- 商品上架 / 編輯 / 刪除
- 管理商品分類與系列
- 訂單與會員資訊管理
